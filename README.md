# Prediksi Harga Rumah - Data Mining Praktikum 13

Proyek ini merupakan implementasi model machine learning untuk memprediksi harga rumah di wilayah Jabodetabek.

## 📊 Dataset
- **Sumber**: `jabodetabek_house_price.csv`
- **Lokasi**: Jakarta, Bogor, Depok, Tangerang, Bekasi
- **Fitur**: Luas tanah, luas bangunan, jumlah kamar, lokasi, dll.

## 🚀 Fitur
- Prediksi harga rumah berdasarkan input pengguna
- Model machine learning yang telah dilatih (model.pkl)
- Antarmuka web sederhana menggunakan Flask (app.py)

## 📁 Struktur Proyek
- app.py # Aplikasi web Flask
- model.pkl # Model machine learning yang sudah dilatih
- district_mean_price.pkl # Data rata-rata harga per wilayah
- jabodetabek_house_price.csv # Dataset harga rumah
- requirements.txt # Daftar dependencies
- ML_meetTerakhir_asdos.ipynb # Notebook eksperimen
- README.md # Dokumentasi proyek
